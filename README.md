# LocalMarks

A static bookmark viewer for your personal `bookmarks.json` database.

## Structure

```
LocalMarks/
├── javascript/
│   ├── script.js      # Main bookmark viewer logic
│   └── info.js        # Database stats page logic
├── stylesheet/
│   └── style.css      # Shared styles for all pages
├── bookmarks.json     # Your bookmark database (generated by book_to_json.py)
├── index.html         # Bookmark browser
├── info.html          # Database stats & domain info
├── favicon.ico
└── README.md
```

## Usage

1. Run `book_to_json.py` to generate `bookmarks.json` from your `.txt` bookmark files
2. Place `bookmarks.json` next to `index.html`
3. Serve the folder with any local server, e.g.:
   ```
   python3 -m http.server 8000
   ```
4. Open `http://localhost:8000` in your browser

## Features

- **Browse** bookmarks by category via the sidebar
- **Search** across title, description, tags, and URL (`Ctrl+K`)
- **Filter by tags** — click any tag pill, multi-select supported
- **Click a tag on a card** to instantly filter the panel by that tag
- **Info page** — stats, category chart, tag cloud, domain grid
- Duplicate URLs are automatically hidden
- Favicons loaded via Google's favicon service
